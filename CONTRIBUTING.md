# 贡献指南

感谢您考虑为本项目做出贡献！请遵循以下步骤：

1. 进行分支操作  
   在提交任何更改之前，请从主分支创建一个新的功能分支：
   ```bash
   git checkout -b my-feature-branch
   ```

2. 进行更改  
   在功能分支上进行更改。

3. 提交更改  
   提交您的更改并写下描述性信息：
   ```bash
   git commit -m '添加了新的特性'
   ```

4. 推送分支  
   将您的功能分支推送到您的fork：
   ```bash
   git push origin my-feature-branch
   ```

5. 提交请求  
   在GitHub上打开一个Pull Request。
